# IconFontAwesome

Mit Hilfe der **IconFontAwesome**-Komponente können Icons aus der FontAwesome-Bibliothek an beliebigen Positionen erzeugt werden. Die Ausgabe des Icons kann über Attribute gesteuert werden und erfolgt barrierefrei. Die Ausgabe erfolgt standardmäßig als **_inline_**-Element.

## Konstruktion

### Code

```tsx
<div>
	<kol-icon-font-awesome _aria-label="" prefix="fas" _icon="home"></kol-icon-font-awesome>
</div>
```

### Beispiel

<div style={{ fontSize: '48px' }}>
  <kol-icon-font-awesome _aria-label="" _prefix="fas" _icon="home"></kol-icon-font-awesome>
</div>

## Verwendung

### Icon

Legen Sie über das Attribut `_prefix` fest, aus welcher Bibliothek das Icon verwendet werden soll. Verwenden Sie außerdem das Attribut `_icon`, um das anzuzeigende Icon festzulegen.
